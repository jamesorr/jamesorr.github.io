Some software that I contribute to
  1. mocsy - improved routines to model the ocean carbonate system (Fortran, usable in Python)
    • publication
    • code
  2. fastspline - the fastest cubic spline in the west (Weinart, 2009) written in MATLAB was converted to Fortran (and is usable in Python)
    • code
  3. seacarb - computes seawater carbonate system parameters (R)
    • CRAN archive
    • GitHub archive
  4. errors - routine to propagate uncertainties for the marine CO2 system
    • available as an extra sheet in to CO2SYS-Excel (used in Excel)
    • available as add-on to CO2SYS-MATLAB (used in MATLAB or Octave)
    • also available within mocsy and seacarb packages
  5. derivnum - routine that calculates sensitivities (buffer factors) for the marine CO2 system
    • available as add-on to CO2SYS-MATLAB (used in MATLAB or Octave)
    • also available within the mocsy and seacarb packages
